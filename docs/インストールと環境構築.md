# インストールと環境構築


## 開発エディター

`Visual Studio 2026` を使います。  


## コンソールアプリで［依存性の注入］などのサービスを使えるようにします

NuGet を使って以下のパッケージをインストールしてください。  

* `Microsoft.Extensions.Hosting` - ホストビルダーを提供するパッケージです。これにより、依存性の注入やロギングなどの機能を簡単に利用できるようになります。


## アプリケーション設定ファイル

Microsoft 公式では TOML 形式の設定ファイルはまだサポートされていないため、JSON 形式の設定ファイルを使用します。  
ファイル名は `appsettings.json` としてください。  

NuGet を使って以下のパッケージをインストールしてください。  

* `Microsoft.Extensions.Configuration` - アプリケーションの設定を管理するためのパッケージです。
* `Microsoft.Extensions.Configuration.Json` - JSON 形式の設定ファイルを読み込むためのパッケージです。


## ロギング

NuGet を使って以下のパッケージをインストールしてください。  

* `Serilog.Extensions.Hosting` - `Microsoft.Extensions.Hosting` と Serilog を統合するためのパッケージです。
* `Serilog.Sinks.Console` - コンソールにログを出力するためのシンクです。
* `Serilog.Sinks.File` - ファイルにログを出力するためのシンクです。
* `Serilog.Settings.Configuration` - 設定ファイルから Serilog を構成するためのパッケージです。

Serilog の設定：  

📄 `appsettings.json` ファイルの `Serilog` セクションに、Serilog の設定を記述します。  

* 📖 [serilog-settings-configuration](https://github.com/serilog/serilog-settings-configuration) - Serilog の設定ファイルの書き方（＾～＾）
