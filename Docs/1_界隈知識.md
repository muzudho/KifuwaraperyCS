# 界隈知識

このプロジェクトは、［コンピューター将棋の思考エンジン］です。  

［コンピューター将棋の思考エンジン］とは何かを、順を追って説明します。  


## [KNOW_01] 広義の将棋と、狭義の将棋

将棋と名前の付いているボードゲームは沢山あります。  

［広義の将棋］の例：  

* はさみ将棋
* 将棋崩し
* 天竺大将棋
* 軍人将棋
* 本将棋
* 詰将棋
* ……などなど。

ここで、［狭義の将棋］は１つしかありません。  

［狭義の将棋］の例：  

* `本将棋` - 単に「将棋」といえば、これを指すことが多いです。


## [KNOW_02] 本将棋

［本将棋］の定義は、 [日本将棋連盟](https://www.shogi.or.jp/)によって[対局規則](https://www.shogi.or.jp/match/taikyoku_rules/)が定められているのではっきりしています。  
もっと柔らかい［本将棋］の説明は [本将棋](https://www.shogi.or.jp/knowledge/shogi/01.html)にあります。  

俗な説明をすると、  
他の［広義の将棋］と比べて、［取った相手の駒を盤の上に置いて使ってよい］というルールを持っているのが［本将棋］の特徴です。  


### [KNOW_01] 日本将棋連盟が定義するものが本将棋

ボードゲームである［将棋］には、類まれなことに、プロを専業とする団体である [日本将棋連盟](https://www.shogi.or.jp/) が存在します。  

例えば［チェス］では、チェスのプレイヤーは、他の仕事と掛け持ちしつつ、空いた時間にチェスの勉強して、[日本チェス連盟](https://japanchess.org/)といった、各国に１つある国を代表するチェス協会に所属して、
自分で参加費も交通費も払って世界大会に出る、といった感じです。  

この日本将棋連盟は、いわゆる [本将棋](https://www.shogi.or.jp/knowledge/shogi/01.html) の団体です。  


#### [PLAN_02]

本将棋を［狭義の将棋］とするなら、本将棋以外の［広義の将棋］も、Wikipedia で１０種類以上羅列されるように、いろいろな種類があります。  

* 小将棋 - 本将棋に似ているが、持ち駒が使えないと考えられる。
* [中将棋／日本中将棋連盟](https://www.chushogi-renmei.com/)
* 大将棋

そして、これらの［広義の将棋］とは別に、ローカル・ルールを取り入れたものを［変則将棋］と呼びます。  
本将棋に変則将棋があるように、中将棋にも変則将棋があります。  
［広義の将棋］と［変則将棋］は別の概念であることに注意してください。
